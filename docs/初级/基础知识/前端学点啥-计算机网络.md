网域名称简称域名，如 googole.com，域名实际上就是特定对应的 IP 地址的别称，相对于 192.168.xx.xx 这种形式的 IP 地址，域名更为人们所能熟记，通常我们访问的域名地址，最终需要通过域名解析系统 DNS 找到对应的 IP 地址，才能够访问到我们需要访问的内容。

## TCP/IP

我们平台通过互联网访问的一切，都基于一些特定的传输规则和协议，这种规则和协议就是 TCP/IP 协议
